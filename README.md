# 👜 Myntra Clone (HTML + CSS)

A front-end clone of the *Myntra* fashion website built using *HTML* and *CSS*, designed to replicate the sleek and trendy UI of a real e-commerce platform.

---

## 🚀 Features

- 👗 Stylish Myntra-inspired homepage layout  
- 📦 Product section with hover effects and grid display  
- 📱 Fully *responsive design* for mobile and desktop  
- 🎨 Clean UI with proper alignment, spacing, and colors  
- 💡 Easy-to-read, well-structured HTML and CSS files  

---

## 🧠 What I Learned

- Using *Flexbox* and *Grid* for layout design  
- Creating *responsive websites* using media queries  
- Maintaining clean, reusable CSS code  
- Understanding front-end structure of e-commerce platforms
